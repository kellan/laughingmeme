CVS -> RSS

cvs2rss.pl is a Perl script (and a couple of modules) that can mine an RSS feed out of a local cvs checkout. (though mostly it asks cvs2cl.pl to do the heavy lifting) Its kind of a hack, but its a cool hack, and I put sometime into thinking about what should values should be in fields like ‘title’ and ‘link’, and it works pretty well.

CVS & the Web

Currently cvs2rss.pl depends on you to have a copy of ViewCVS(aka, the CVS browser on Sourceforge) running somewhere public for it to link to, but is designed to be extended for use with other web repository browsers. An example CVS RSS feed for XML::RSS.
